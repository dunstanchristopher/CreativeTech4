Digital Twins

ErNoFa—A Blended Pop Star Made With Machine Learning

The Brief: Working in a team of 3, we created a digital twin model(s) using one another as our datasets and through various machine learning software examples. Using ourselves as a medium, how can we use machine learning to extend one another’s attributes, mix, mash, and collaborate and formulate new in-between aesthetics from these various processes?

Using a machine learning program calling RunwayML, our team decieded to use the Liquid Warping GAN as the basis for our porject. This machine learning program takes the movement of one reference and applies it to the second reference.
